== Description ==
含百度站长、360站长、Bing站长、今日头条站长、tag标签内链、百度地图sitemap、关键词排名查询监控、360站长JS自动推送、 网站蜘蛛、robots、图片alt标签、天级推送、category隐藏、死链查询、百度自动推送、批量提交URL到站长、百度收录查询、文章原创率检测、批量推送未收录、301/404等功能、AI文章助手、流量监控。

1、首页关键词

因为wordpress只有title网站标题的设定，没有keywords,description的设定，也就是关键词和描述。特地增加该功能增加权重，及领域区分。

2、百度主动推送

通过插件的配置百度主动推送准入密钥，有批量推送，以及定时自动批量推送的功能，无需每天人工主动提交URL链接的烦恼。

3、自动推送

通过开关按钮来配置百度主动推送功能。自动推送是百度搜索资源平台为提高站点新增网页发现速度推出的工具，安装自动推送JS代码的网页，在页面被访问时，页面URL将立即被推送给百度。

4、Sitemap推送

Sitemap推送功能也是百度站长URL提交功能之一，自动读取文章、列表、其他页面等HTML的页面进行生成SITEMAP，并且会根据文章新旧程度定级文章的权重，最大是1.0分配的是首页，0.9是最新的文章，其他陈旧文章0.5以此类推，也会生成HTML和XML的地图方便博主快速复制Sitemap地址，到百度站长平台提交sitemap地址，省去人工生成更新SITEMAP时间，提高优化效率。

5、图片Alt标签填充

主要是利于网站优化的一种，因为图片没有Alt标签会被搜索引擎对网站减分降低权重，因为我们不可能每个图片都可以去添加Alt标签，所以该功能可以大大降低人工成本，将Alt标签自动读取标题及相关的关键词，可以批量优化，减少人工添加的烦恼，从而提升网站权重。

6、站内内链

做网站优化的小伙伴都知道，内链为王，外链为皇，内链也是十分重要的，所以小编根据这个开发了插件的这一项功能，可以根据提供指定的关键词，自动添加内链，一键批量添加，面对几千篇的文章无需再人工添加及更改。

7、Robots.txt生成

这个主要用于引导搜索引擎蜘蛛，小编曾经维护过一个权重6的网站，因为国外的蜘蛛过于庞大，当时5M的带宽50%的资源被其他搜索引擎占用，不得不限制蜘蛛，所以说有些人认为是流量攻击的胡话就先看看自己是否被国外蜘蛛盯上，开启了这个Robots.txt类似于黑名单可以拉黑不让来的蜘蛛。还有其他功效需要自己了解

8、301重向&404页面

301重向，一般都是不会设置服务器的用户，主要功能也是把不带WWW的域名指向带WWW的域名，增加权重，否则会出现2个网站的不同权重的情况，这个新手经常遇到。不带WWW的权重比带WWW的要高，而且收录上面也很多不带WWW的域名会多点。

404页面，基本上优化站点必备的，也是给不会配置服务器的用户使用，基本上都是一键操作十分便捷，主要是提升网站权重和质量，引导蜘蛛等。

9、天级提交

熊掌号的天级推送，配额每个账号只有10条，随着指数越高配额越高，但是小编把插件设置的是在文章列表也有提交按钮，点击即可提交自己想要提交的文章，省去登录后台的烦恼，大大提高工作效率。

10、一键查询网站所有URL链接的收录情况，并且可以一键推送到熊掌号、站长号，未收录的URL链接。（后期会开发定时自动推送未收录的URL）

11、sitemap一键生成，自动更新，含有地图HTML版、XML版。

12、增加功能网站蜘蛛，可以实时监控什么蜘蛛爬取了什么页面，返回状态代码是多少，无需查看服务器日志。

13、死链查询功能，查询网站内所有URL链接是否有死链，如果发现生成silian.xml文件提交百度站长即可。

14、增加关键词排名查询监控系统，可以实时监控关键词排名多少，无需在借助其他工具查询。

15、增加tag标签内链添加功能，可以指定关键词让所有文章内包含该关键词的字段拥有超链。该功能是有两种，指定跳转超链，或者自动跳转标签页，这是SEO高阶玩家的必备神器！

16、增加category字段隐藏，更好的优化URL链接。

17、增加分类栏目的SEO标题，关键词，描述SEO填写。

18、增加流量词获取，可以查看哪些关键词有排名。

19、增加360站长、Bing站长、今日头条站长、神马站长

20、增加文章原创率检测。

21、智能改写。

22、快速排名。

== Installation ==

方式1：在线安装(推荐)
1. 进入WordPress仪表盘，点击“插件-安装插件”，关键词搜索“百度站长SEO合集”，找搜索结果中找到“百度站长SEO合集”插件，点击“现在安装”；
2. 安装完毕后，启用 `百度站长SEO合集` 插件.
3. 通过“设置”->“百度站长SEO合集” 进入插件设置界面.
4. 至此，该插件安装完毕。

方式2：上传安装

FTP上传安装
1. 解压插件压缩包seo.zip，将解压获得文件夹上传至wordpress安装目录下的 `/wp-content/plugins/`目录.
2. 访问WordPress仪表盘，进入“插件”-“已安装插件”，在插件列表中找到“百度站长SEO合集”，点击“启用”.
3. 通过“设置”->“百度站长SEO合集” 进入插件设置界面.
4. 至此，该插件安装完毕。

仪表盘上传安装
1. 进入WordPress仪表盘，点击“插件-安装插件”；
2. 点击界面左上方的“上传按钮”，选择本地提前下载好的插件压缩包seo.zip，点击“现在安装”；
3. 安装完毕后，启用 `百度站长SEO合集` 插件；
4. 通过“设置”->“百度站长SEO合集” 进入插件设置界面.
5. 至此，该插件安装完毕。

== Frequently Asked Questions ==

= 百度主动推送密钥在哪里获取？ = 
主要分为，站长密钥，熊掌号密钥，需要分别登陆百度站长及熊掌号获取。

= 百度熊掌id的天级推送-数据反馈记录为0，请问这是什么原因导致的？ = 
您好，这一般是由于百度搜索资源平台内部更新导致。数据存在更新延迟的问题，请耐心等待百度方面完成更新工作。反馈数据仅是针对线上数据的统计；线上的收录、展现、排名，不会受此影响！

= 百度搜索资源平台PC推送密钥验证提示site sid is empty或者接口请求错误 =
这是百度搜索资源平台密钥验证服务器的问题。如提示site sid is empty或者接口请求错误时，点击重试或者稍后再试即可。

= 百度搜索推送插件推送设置推送提示超额 =
百度搜索资源平台目前URL更新/删除接口均无效，提示超额。如果你的百度搜索推送插件出现超额提示，请更新到插件最新版本，已经临时将URL更新和删除接口改为URL新增接口推送。

= SEO合集插件是否可以生成网站Sitemap? =
不仅可以生成SITEMAP而且分别会给出2个版本，一个是HTML，一个是XML

= 是否使用了SEO合集插件，即可保证百度搜索正常收录页面? =
不是的。百度搜索推送管理插件仅是帮助站长将WordPress博客更新的内容快速推送给百度搜索，以便于百度搜索快速地发现URL。但百度搜索是否收录及收录时效，由百度决定。但可以肯定的是，该插件是有利于做百度搜索优化的。

= 为什么插件的主动及自动推送数据统计与百度搜索资源平台后台显示的数据不一致? =
两者的数据肯定不一致。插件展示的数据仅代表插件协作推送至百度搜索资源平台的数据，即完整推送数据；百度搜索资源平台的统计数据为自动/主动/sitemap三种推送方式去重数据，除sitemap推送外，自动及主动推送不重复计算已经收录或者推送过的数据；推送数据不代表收录数据，积极向百度推送数据目的是为了更好地获得收录数据。

= 为什么插件的收录统计与百度搜索引擎查询到的数据不一致？ = 
插件提供的百度收录统计数据为估算值，数据每小时更新一次。网站管理员如需了解更准确的索引量，请使用百度站长平台。

= 是否使用了插件的熊掌id天级推送数据，数据就一定被百度收录？ = 
不一定。但经过长时间的数据积累，熊掌id天级推送是目前推送给百度搜索引擎的最佳方式，持续使用熊掌id天级推送高质量的原创数据，能够大幅度提升网站的百度收录效率。切记三个要点：高质量内容；天级配额用完；每天持续更新。

= 为什么文章百度收录查询有时候不准确？ = 
文章百度收录状态仅供参考，实际收录情况以百度搜索为准。

= 为什么Sitemap推送设置无法检测到Sitemap地址？ = 
我们兼容了大部分主流的sitemap插件，有可能无法检测一些插件生成的sitemap地址。也有可能你未安装sitemap插件或者无法检测到你所安装插件所生成的sitemap文件。
